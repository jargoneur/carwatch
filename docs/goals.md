# Goals achieved & missed

## Achieved

- Kernfunktionen erfolgreich umgesetzt: Übersichtliche Zusammenführung von Autoanzeigen aus einer externen Automobilseite an einem Ort
- Funktionierende Filter
- Übersichtliche Anzeige der Autos
- reibungslose Anmelden von Konten

## Missed

- Einbindung mehrerer externer Automobil-Seiten
- Eventuelles Anschreiben von den Autoanzeigen
- genaue Preisangaben bei den Anzeigen
