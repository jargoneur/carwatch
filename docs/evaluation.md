# User evaluation

- Direktes Feedback zur Verständlichkeit und Bedienung
- Kurzer Test mit Nutzern zur Bewertung von Übersichtlichkeit, Filtern und Scoring-Verständlichkeit
- Feedback zeigte schnellen Mehrwert, aber Verbesserungsbedarf bei Erklärung des Scoring-Systems
