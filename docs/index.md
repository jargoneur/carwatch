# Carwatch

## Value Proposition

- Zusammenführung von Autoanzeigen aus externen Automobilseiten an einem Ort: spart Zeit und reduziert Suchaufwand.
- Transparente Vergleichbarkeit durch standardisiertes Scoring-System von **0.00 bis 100.00**.
- Effizientere Fahrzeugsuche durch Filter nach relevanten Kriterien statt unübersichtlicher Einzelportale.
