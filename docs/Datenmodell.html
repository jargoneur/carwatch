Fahrzeug – Datenmodell

Diese Seite beschreibt ausschließlich die Datenstruktur eines Fahrzeugs. Es werden keine realen Daten verwendet.

Feld Typ Pflicht Beschreibung Werte
id integer ja Eindeutige Fahrzeug-ID —
marke string ja Automarke —
modell string ja Modellbezeichnung —
baujahr integer ja Jahr der Erstzulassung —
zustand enum ja Zustand des Fahrzeugs neu, sehr_gut, gut, ok, schlecht
kilometer integer ja Kilometerstand (km) —
motortyp enum ja Motortyp benzin, diesel, hybrid, elektrisch
farbe string ja Fahrzeugfarbe —
score integer nein (berechnet) Bewertungs-Score, abgeleitet aus Baujahr, Zustand, Kilometer und Motortyp 0–100
```
